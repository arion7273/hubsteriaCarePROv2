# HIPAA security readiness checklist

HubsteriaCarePRO must complete these controls before production use with PHI.

## Administrative safeguards

- Assign security officer and privacy officer.
- Complete workforce access policy.
- Define sanction policy for inappropriate access.
- Maintain business associate agreement inventory.
- Complete incident response procedures.
- Document disaster recovery and emergency mode operations.

## Technical safeguards

- Enforce unique user identity.
- Require MFA for administrative and clinical access.
- Enforce role-based access control.
- Enforce tenant isolation on every server-side request.
- Encrypt data in transit with TLS.
- Encrypt PHI at rest.
- Store secrets in a managed secret store.
- Maintain immutable audit logs.
- Add automatic logoff/session expiration.
- Add integrity controls for clinical and financial records.

## Physical and operational safeguards

- Validate cloud hosting controls.
- Restrict production console access.
- Review backup storage access.
- Document media disposal and data retention requirements.

## Validation gates

- HIPAA security risk assessment complete.
- Penetration test complete.
- Backup restore test complete.
- Disaster recovery drill complete.
- Audit log validation complete.
- Incident response tabletop complete.
